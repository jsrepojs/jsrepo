# @jsrepo/autoversion

A package for automating the versioning of jsrepo registries.

Based on how you change your registry `@jsrepo/autoversion` will automatically update the version of your registry.

**Breaking changes:**
- Remove a file from a registry item

**Minor changes:**
- Remove a registry item
- Add a file to a registry item
- Add dependencies to a registry item
- Remove dependencies from a registry item

**Patch changes:**
- Update the code for a registry item
- Update the dependencies of a registry item
